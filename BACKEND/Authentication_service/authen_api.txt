## ðŸ”¸ **POST APIs (Postman Required)**

### 5. **Register User**
```
Method: POST
Local:   http://localhost:9091/api/register
Gateway: http://localhost:8305/api/register
Headers: Content-Type: application/json
Body (raw JSON):
{
  "email": "user@email.com",
  "password": "password123",
  "fullName": "John Doe",
  "role": "USER",
  "phone": "1234567890"
}
Comment: Creates new user account. Role can be "USER" or "DRIVER"
```

### 6. **Login User**
```
Method: POST
Local:   http://localhost:9091/api/login
Gateway: http://localhost:8305/api/login
Headers: Content-Type: application/json
Body (raw JSON):
{
  "email": "user@email.com",
  "password": "password123"
}
Comment: Authenticates user and returns JWT token
```
